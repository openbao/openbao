# version

The `version` command prints the OpenBao version:

```shell-session
$ bao version
OpenBao v1.2.3
```

It can also be printed by adding the flags `--version` or `-v` to the `bao` command:

```shell-session
$ bao -v
OpenBao v1.2.3, built 2022-05-03T08:34:11Z
```
