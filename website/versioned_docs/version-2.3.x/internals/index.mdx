# OpenBao internals

This section covers the internals of OpenBao and explains the technical
details of how OpenBao functions, its architecture and security properties.

**Note:** Knowledge of OpenBao internals is recommended but not required to use OpenBao. However, if you're operating OpenBao, we recommend understanding the internals.
